## 🚀 Quick start

1.  **Clone the repo.**

    Use the Gatsby CLI to create a new site, specifying this repo.

    ```sh
    # create a new Gatsby site using the default starter
    gatsby new site-folder-name https://github.com/codedrips/gatsby-starter-codedrips
    ```
